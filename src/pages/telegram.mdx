# Telegram

<Note>
Create your first Telegram bot if you don't have one yet [create your firts Telegram bot](https://core.telegram.org/bots/tutorial "Create your firts Telegra bot")
</Note>

To implement a chatbot on Telegram, you need to install the library `@builderbot-plugins/telegram`

## Install

```bash {{ title: 'npm' }}
    npm install @builderbot-plugins/telegram
```

```typescript
    import { createBot, MemoryDB, createProvider, addKeyword, createFlow, EVENTS } from '@bot-whatsapp/bot'
    import { createShopifyFlow } from '@builderbot-plugins/shopify'
    import { TelegramProvider } from '@builderbot-plugins/telegram'

    const main = async () => {
        const provider = createProvider(TelegramProvider, {
            token: 'YOUR_TOKEN_HERE'
        })

        const { flow } = await createShopifyFlow()

        await createBot({
            database: new MemoryDB(),
            provider,
            flow: createFlow(flow)
        })

    }
    main()
```

## Flows
Create your first flow with Telegram.

```ts
    /* code stuff */

    const provider = createProvider(TelegramProvider, {
            token: 'YOUR_TOKEN_HERE'
    })
    const welcome = addKeyword(EVENTS.WELCOME).addAnswer('Hi, Welcome to Telegram!');

    await createBot({
        database: new MemoryDB(),
        provider,
        flow: createFlow([welcome])
    })

    /* code stuff */
```

## Webhook

Alright, let's continue. If you need to expose a port or an API for listening to send messages, you can do it as follows.

```typescript
    import 'dotenv/config'
    import { createBot, MemoryDB, createProvider, addKeyword, createFlow, EVENTS } from '@bot-whatsapp/bot'
    import { createShopifyFlow } from '@builderbot-plugins/shopify'
    import { TelegramProvider, handleCtx } from '@builderbot-plugins/telegram'

    const main = async () => {

        const provider = createProvider(TelegramProvider, {
            token: "YOUR_TELEGRAM_TOKEN_HERE"
        })

        provider.initHttpServer(9000)
            .http.server.post('/webhook', handleCtx((null, req, res) => {
                    console.log('I am function what doing someting')
                    res.end('ok')
            }))
        
        const welcomeFlow = addKeyword('Hi!!').addAnswer('Hi!, friend.')

        await createBot({
            database: new MemoryDB(),
            provider,
            flow: createFlow([welcomeFlow])
        })

    }

    main()
```
