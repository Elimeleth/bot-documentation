# Context

O objeto "contexto" é um componente-chave dentro da estrutura que fornece aos desenvolvedores acesso a informações essenciais sobre a interação contínua entre o Usuário e o bot. Esta informação inclui propriedades predefinidas como `from` (representando o remetente) e `body` (contendo a mensagem ou entrada do utilizador).

Ao utilizar a estrutura [addAnswer](/add-functions#add-keyword) and [addAction](/add-functions#add-action) métodos, o objeto de contexto é passado como o primeiro argumento nas funções de retorno de chamada. Isso permite que os desenvolvedores acessem e manipulem as propriedades de contextos dinamicamente, permitindo respostas e ações personalizadas com base nas especificidades de cada interação do Usuário.

<CodeGroup>

```ts {{ title: 'flow-addKeyword.ts' }}
  import { addKeyword } from '@builderbot/bot';

  const flowA = addKeyword('register')
    .addAnswer('What is your name?', { capture: true }, async (ctx, { state }) => {
      const responseName = ctx.body
      const nameFrom = ctx.name
      const numberFrom = ctx.from

      console.log(`Other properties:`, ctx)
    })

  export default flowA
```
```ts {{ title: 'flow-addAction.ts' }}
  import { addKeyword } from '@builderbot/bot';

  const flowB = addKeyword('resume')
    .addAction(async (ctx, { state }) => {
      const responseName = ctx.body
      const nameFrom = ctx.name
      const numberFrom = ctx.from

      console.log(`Other properties:`, ctx)
    })

  export default flowB
```
</CodeGroup>

É importante notar que as propriedades que você sempre terá disponíveis são de, body, name, mas você também terá muito mais propriedades diferentes dependendo do [provider](/concepts#provider).
