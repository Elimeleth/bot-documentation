# Groups

Os grupos são onde as comunidades vivem em Protocolo-eles são uma coleção de contatos com os quais você está conversando de uma só vez. Nesta página, vamos nos aprofundar nos diferentes endpoints de grupo que você pode usar para gerenciar grupos programaticamente. Veremos como consultar, criar, atualizar e excluir grupos. {{className: 'lead'}}

## The group model

O modelo de grupo contém todas as informações sobre seus grupos, incluindo quais contatos estão no grupo e o nome, a descrição e o avatar do grupo.

### Properties

<Properties>
  <Property name="id" type="string">
Identificador único para o grupo.
  </Property>
  <Property name="name" type="string">
O nome do grupo.
  </Property>
  <Property name="description" type="string">
A descrição do grupo.
  </Property>
  <Property name="avatar_url" type="string">
O URL da imagem do avatar para o grupo.
  </Property>
  <Property name="conversation_id" type="string">
Identificador único para a conversa que pertence ao grupo.
  </Property>
  <Property name="contacts" type="array">
Uma matriz de objetos de contato que são membros do grupo.
  </Property>
  <Property name="created_at" type="timestamp">
Data e hora de quando o grupo foi criado.
  </Property>
  <Property name="archived_at" type="timestamp">
Data e hora de quando o grupo foi arquivado.
  </Property>
</Properties>
